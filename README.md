# 💫 About Me:
Привіт! Я розробник, який фокусується на створенні системних інструментів, високопродуктивного софту та інтелектуальних помічників. <br>Мій основний стек включає Rust, Python та C++.

Ось над чим я працюю:

Rust → розробка швидких утиліт (наразі працюю над власним архіватором) та системного ПЗ.<br>
Python → створення AI-асистентів, автоматизація та бекенд (FastAPI/Django).<br>
C++ → розробка продуктивних застосунків та вивчення низькорівневих процесів.<br>
TypeScript + React → розробка сучасних інтерфейсів (Electron, Web).<br>
