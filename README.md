# 💫 About Me:
Hi there! I'm a software developer focused on building high-performance systems, intuitive GUIs, and AI-driven solutions. I enjoy diving into low-level logic with Rust and C++, while also building modern, scalable applications with Python and TypeScript.

**What I'm currently working on:**
* **Rust & C++:** Crafting efficient system utilities and performance-critical software. I'm especially interested in custom file compression and building modern GUIs with Rust.
* **AI & Automation:** Developing intelligent personal assistants (like my JARVIS project) and automating complex workflows using Python.
* **Web & Desktop:** Building clean, functional user interfaces using React and Electron.
